## Overview

- Displays the School Programs section which contains information on educational programs based on education level.
- Presenting the section title “Learning Programs for Every Stage” as the main introduction.
- Provides a brief description of the program's hands-on learning approach.
- Provides a list of program categories based on education levels such as Preschool, Primary School, Secondary School, and All Ages.
- Use checklist elements to display program categories clearly and easily understood.
- Displays visuals of children's activities as a representation of educational program activities.

## User Story
