## Overview

- Displays the condition when no results are found for the keyword entered by the user.
- Provide an informative message such as “No results found” to inform the user that the search was unsuccessful.
- Provides users with suggestions to check spelling or try another keyword.
- Display a display that remains consistent with the page design so as not to confuse users.

## User Story

<Accordion title="View User Story" defaultOpen>
  - **As a user**, I want to know that no results were found so I don't get confused by a blank display.
  - **As a user**, I want to get suggestions to improve my search so I can try other keywords.
  - **As a user**, I want to keep the display clear and informative so that the experience remains comfortable even if there are no results.
