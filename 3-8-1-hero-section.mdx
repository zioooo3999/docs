## Overview

- Displaying the hero section as the opening page for the School Program Article with the main visual in the form of children's educational program activities at Tempe Park.
- Provide a program title that is displayed clearly and prominently as the main identity of the page.
- Display breadcrumb navigation (Education → School Program → Detail) to help users understand the page structure and navigation.
- Using warm and interactive visuals to provide an educational, fun, and family-friendly feel.
- Provide a strong first impression so that users are interested in continuing to explore further program information.

## User Story

<Accordion title="View User Story" defaultOpen>
  - **As a visitor**, I want to see the main image of the program so I can get a visual idea of ​​the activities that will be carried out.
  - **As a visitor**, I want to see the program title clearly so I know what program I am opening.
  - **As a visitor**, I want to see breadcrumb navigation so I know where I am on the current page and can return to previous pages easily.
  - **As a visitor**, I want to get an interesting first impression from the appearance of the page so that I am interested in exploring further information.
</Accordion>
