## Overview

- Displays the Vision & Mission section as an explanation of Tempe Park's long-term direction and goals.
- Provides two main sections, namely “Our Vision” and “Our Mission” which are displayed separately but are interrelated.
- Features a vision description that explains Tempe Park's aspirations as a center for tempeh-based innovation and education.
- Displays a mission description that explains the strategic steps in realizing the vision.
- Use a dynamic visual layout with a combination of text and images to increase appeal.
