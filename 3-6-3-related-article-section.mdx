## Overview

- Displays the Related Article section as a recommendation for other articles that are relevant to the article currently being read.
- Provide clear section titles (“Related Article”) to indicate the relationship to the main content.
- Displays a list of articles in the form of visual cards containing thumbnails, categories, titles, and short descriptions.
- Provide a “Read More” CTA on each card to direct users to the related article detail page.
- Use a grid layout (multiple columns) to display multiple articles at once neatly.
- Displays category labels on each card to help users understand the topic of the article.

## User Story

<Accordion title="View User Story" defaultOpen>
  - **As a user** reading an article, I want to see recommendations for other relevant articles, so I can find additional content that interests me.
  - **As a user**, I want to see articles in the form of informative visual cards, so I can quickly understand the content of the article before opening it.
  - **As a user**, I want to press the “Read More” button on an article, so I can immediately read the full article.
  - **As a user**, I want to see the article categories on each card, so I can select articles based on the topics I'm interested in.
</Accordion>
