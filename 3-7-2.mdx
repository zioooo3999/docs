## Overview

- Featuring an educational section about the history of tempeh as the main introduction on the Education page.
- Presenting information in the form of cards divided into several topics such as origins, development across generations, and the global influence of tempeh.
- Display illustrative images on each card to reinforce visuals and aid user understanding.
- Provide a title and short description on each card so that the information is easy to understand and not too long.
- Using a grid/card-based layout to improve readability and user comfort in browsing content.

## User Story

<Accordion title="View User Story" defaultOpen>
  - **As a visitor**, I want to know the history of tempeh in a concise and interesting way, so that I can understand its origins and cultural value.
  - **As a visitor**, I want to see information presented in visual form (images and cards), so it is easier to understand than long text.
  - **As a visitor**, I wanted to know how tempeh evolved from a traditional food to a global product, so that I could gain broader insights.
  - **As a visitor**, I want to read content that is short but informative, so that I don't get bored while learning about the history.
