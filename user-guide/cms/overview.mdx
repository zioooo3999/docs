Tempe Park's Content Management System (CMS) is a Strapi-based content management system used to manage all information and digital content on the Tempe Park platform. This CMS allows administrators to centrally manage data without requiring direct changes to the application code. This system is designed to make content management faster, more structured, and more efficient.

By utilizing Strapi as a headless CMS, Tempe Park can manage various types of data such as website pages, articles, promotional banners, tickets, events, galleries, and other supporting information through an easy-to-use interface.
