## Overview

- Displays a list of available job openings in Tempe Park.
- Provides search features based on job name, keywords, skills, or location.
- Provides job filters based on Job Level, Job Type, and Location.
- Displays summary vacancy information such as position name, location, job category, and application deadline.
- Provides a See Detail button to view complete information about the job vacancy.
- Provides an Apply button to apply for the selected position.
- Provides pagination features to make it easier for users to navigate the list of available vacancies.
- Displaying a hero banner as an identity and introduction to the career page.

## User Story

<Accordion title="View User Story" defaultOpen>
  - **As a job seeker**, I would like to see a list of available job openings, so I can find out what positions Tempe Park is currently opening.
  - **As a job seeker**, I want to search for openings based on keywords, skills, or location, so I can find relevant jobs more quickly.
  - **As a job seeker**, I want to filter vacancies by job level, job type, and location, so that the search results are more relevant to my needs.
  - **As a job seeker**, I want to see detailed information about each vacancy, so I can understand the qualifications, responsibilities, and requirements of the position before applying.
  - **As a job seeker**, I would like to apply directly through the Apply button, so that the registration process can be done easily and efficiently.
  - **As a job seeker**, I want to know the application deadline for each vacancy, so I can submit my application before the recruitment period ends.
</Accordion>
