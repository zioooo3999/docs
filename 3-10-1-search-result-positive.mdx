## Overview

- Displays search results based on keywords entered by the user.
- Provides a list of results in the form of a card/list containing an image, title, category, and short description.
- Use a grid or list layout so that the results are easy for users to scan.
- Displays search keywords as context so users know the results displayed are relevant.
- Provide a CTA such as “Learn More” or click on the card to go to the related detail page.

## User Story

<Accordion title="View User Story" defaultOpen>
  - **As a user**, I want to see relevant search results so I can find the information I'm looking for quickly.
  - **As a user**, I want to see a title and a short description so I can determine which result is the most relevant.
  - **As a user**, I want to click on one of the results so I can see more detailed information.
