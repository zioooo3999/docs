## Overview

- Displays the main headline: “Plan Your Visit” as a page marker and main context for users.
- Using an aerial view of Tempe Park to provide a comprehensive overview of the location and atmosphere.
- Comes with a dark overlay to improve contrast and text readability.
- It has sub-navigation (menu tabs) consisting of: “Buy Ticket”, “Visitor’s Guide”, “Calendar”, and “How to Get Here”.
- Sub-navigation functions as anchor navigation that directs users to related sections on the page.
- The “Buy Ticket” menu is displayed as the default/active state to encourage primary action.
- Keep the global navigation at the top (Home, Plan Your Visit, Experience, etc.) for access to other pages.
- Functions as an entry point that directly directs users to specific needs in planning a visit.

## User Story

<Accordion title="View User Story" defaultOpen>
  - **As a user**, I want to immediately understand that this page contains information for planning a visit to Tempe Park.
  - I want to be able to quickly access important sections such as ticket purchases, guides, schedules, and locations.
  - I want clear navigation so I don't have to scroll long to find the information I need.
  - I want to be directed straight to the main action like buying tickets without having to search manually.
  - **So I can** plan my visit more efficiently and comfortably.
</Accordion>
