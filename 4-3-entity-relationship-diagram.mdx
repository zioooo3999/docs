The following Entity Relationship Diagram (ERD) illustrates the relational structure of the database in the system, including the relationships between key entities such as users, roles, workflows, and content management. Many-to-many relationships are implemented using link tables (\_lnk), while dynamic components use a polymorphic pattern through the \*\_cmps table.
