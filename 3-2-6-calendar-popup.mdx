## Overview

- Appears as a modal popup when the user selects a date or activity in the Calendar section.
- Displays detailed information regarding the selected activity, such as activity time (09:00 – 09:30), activity name (Guided Park Exploration Tour), and location.
- Provides a main visual in the form of an image carousel/slider to provide an overview of the activities to be followed.
- Displays a complete description of the activity to help users understand the experience offered.
- Provides additional navigation such as a Back button to return to the previous calendar view.
- Include a “Buy Ticket” Call-to-Action (CTA) to direct users directly to the ticket purchasing process based on the selected activity.
- Designed to provide more in-depth information without distracting users from the main page (calendar section).

## User Story

<Accordion title="View User Story" defaultOpen>
  - **As a user** who is viewing a schedule of activities on the calendar,
  - **I want to** see the full details of the activity I selected in a popup,
  - **So I can** understand the information about the activity and immediately decide to buy a ticket or go back to see other schedules.
