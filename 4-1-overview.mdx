## Overview

The Tempe Park website's database is designed to support structured, flexible, and easily expandable content management, in line with the system's headless CMS architecture approach. The database serves as the central repository for all data used by the website, including page content, articles, programs, activities, and other supporting information.

The database system used is a relational database (such as PostgreSQL or MySQL) integrated with Strapi as a content management layer. In this case, Strapi acts as an abstraction layer that manages data structures, relationships between entities, and provides an API that will be consumed by the frontend built using next js.
