## Overview

- Displays complete program information details including descriptions, objectives, and activities to be carried out in the program.
- Displays the Program Schedule which contains the dates and times of activities so that visitors know the program implementation schedule.
- Provide additional brief information about the program in the sidebar to strengthen user understanding.
- Display a clear and easily accessible call-to-action button (Buy Ticket) to encourage users to register.

## User Story

<Accordion title="View User Story" defaultOpen>
  - **As a visitor**, I want to read the program details so that I understand the content and benefits of the program.
  - **As a visitor**, I want to see the program schedule so I can know when activities are taking place.
  - **As a visitor**, I want to see additional information in a concise manner so I can get a quick overview of the program.
  - **As a visitor**, I want to see a clear Buy Ticket button so I can easily purchase tickets directly.
</Accordion>
