## Overview

- Displays the Activities section as part of the interactive experience exploration that visitors can do.
- Provide an engaging section title (“Experience Hands-On Moments Made for All Ages”) to emphasize the inclusive and educational experience.
- Displays a list of activities in the form of attractive and informative visual cards.
- Each activity card contains a representative image, the name of the activity (such as Tempe Making, Baking & Cooking, Gardening), and the CTA “Learn More”.

## User Story

<Accordion title="View User Story" defaultOpen>
  - **As a user** who wants to know what's going on at Tempe Park.
  - **I want to** see a list of available activities in an engaging visual format, so I can understand what experiences I can have.
  - **As a user**, I want to see activities that are suitable for different ages, so I can choose the right activity for me or my family.
  - **As a user**, I want to be able to navigate the activity list easily using a slider or arrow keys, so I can see more activity options without difficulty.
</Accordion>
