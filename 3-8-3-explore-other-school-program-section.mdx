## Overview

- Displays the Explore Other School Program section to provide other program recommendations to visitors.
- Provides several program cards containing images, categories (Primary School, Secondary School and Above, All Ages), short descriptions, and CTAs.
- Using a grid layout (3 columns) to display the program list neatly and easily scannable.
- Displays visual images on each card to provide an overview of the activities of each program.
- Provide a “Learn More” button or link on each card to direct users to the related program details page.

## User Story

<Accordion title="View User Story" defaultOpen>
  - **As a visitor**, I would like to see recommendations for other programs so that I have alternative choices besides the program I am currently viewing.
  - **As a visitor**, I want to see pictures and categories of programs so I can quickly understand the differences between each program.
  - **As a visitor**, I want to read a brief description of the program so I can get an initial idea before diving into the details.
  - **As a visitor**, I want to click the “Learn More” button so I can see complete information about the program I am interested in.
</Accordion>
