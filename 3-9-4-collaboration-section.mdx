## Overview

- Featuring the section title “Working Together Across Industries” as an affirmation of cross-sector collaboration.
- Presenting content in the form of a card slider/carousel containing partner or collaborator information.
- Each card displays the partner's logo/identity, title (e.g. brand/initiative name), and a short description.
- Use a horizontal slider layout to display multiple collaborations in one area without taking up too much space.
- Provides a slider indicator (dot navigation) to show the number and position of the active slide.

## User Story

<Accordion title="View User Story" defaultOpen>
  - **As a visitor**, I would like to see information about the collaborations undertaken so that I can understand Tempe Park's network and partners.
  - **As a visitor**, I want to see the partner's logo or identity so I can recognize the parties involved in the collaboration.
  - **As a visitor**, I want to read a short description of each collaboration so that I understand the purpose or form of the collaboration.
  - **As a visitor**, I want to swipe or navigate the slider so I can see more available collaborations.
</Accordion>
