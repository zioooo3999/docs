## Overview

- Display the hero image as the main visual on the content detail page.
- Provide breadcrumb navigation to help users find their way around the page.
- Displays structured titles and content so that it is easy to read.
- Display related products in the sidebar complete with product information and purchase buttons.
- Provides Ticket and Back to Top buttons that can be accessed while the user is scrolling.

## User Story

<Accordion title="View User Story" defaultOpen>
  - **As a user**, I want to see clear titles and content, so I can easily understand the information being conveyed.
  - **As a user**, I want to see breadcrumb navigation, so I can know where I am on the page and move to the previous page quickly.
  - **As a user**, I want to see related products while reading content, so I can find relevant products.
  - **As a user**, I want to easily access product and ticket purchase buttons, so I can perform the desired action without having to search for other menus.
</Accordion>
