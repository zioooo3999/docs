## Overview

- Presenting a collection of menu inspirations in the form of attractive and informative visual cards (image-based layout).
- Showing food images as the main focus to arouse user interest.
- Each card is equipped with a menu name/short title as the dish's identity.
- Provide a “View Ingredients” CTA to allow users to view detailed ingredients or menu composition.

## User Story

<Accordion title="View User Story" defaultOpen>
  - **As a visitor**, I want to see various tempeh-based menu inspirations so I can get interesting food preparation ideas.
  - **As a visitor**, I want to see attractive menu images so that I am interested in learning more about the dishes.
  - **As a visitor**, I want to know the name or type of menu so I can understand the variety of dishes on display.
  - **As a visitor**, I want to press the “View Ingredients” button so I can see the ingredients or details of the menu.
</Accordion>
