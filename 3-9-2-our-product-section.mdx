## Overview

- Displays the section title “Tempe Park’s Curated Collection” as a marker for the available product catalog.
- Provides a category filter panel on the left side to help users filter products by category and subcategory.
- Displays a list of products in a structured and responsive grid card layout.
- Each product card displays the product image, product name, variants (if any), and price.
- Display additional information such as discounts (if available) to attract the user's attention.

## User Story

<Accordion title="View User Story" defaultOpen>
  - **As a visitor**, I want to be able to filter products by category and subcategory so I can quickly find products that suit my needs.
  - **As a visitor**, I want to see a list of products in a neat grid so I can easily compare products visually.
  - **As a visitor**, I want to see information such as the name, price, and product variants so I can understand the product before deciding to buy.
  - **As a visitor**, I want to know if a product is on sale so I can get the best deal.
</Accordion>
