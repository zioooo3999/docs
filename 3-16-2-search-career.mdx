## Overview

- Displays the Career page which serves as an information center for job vacancies available in Tempe Park.
- Provides a job search feature based on keywords, such as position name or field of interest.
- Provides search filters based on candidate category (e.g. Fresh Graduate), job type, and location to make it easier for users to find suitable positions.
- Displays a list of job vacancies that match the selected search criteria and filters.
- Displays key information for each vacancy, such as position name, work location, job category, and application deadline.
- Provides a See Detail button to view complete information about the selected position.
- Provides an Apply button to make it easier for candidates to submit applications directly.
- Using a simple and structured display so that users can find job vacancy information quickly and easily.

## User Story

<Accordion title="View User Story" defaultOpen>
  - **As a job seeker**, I want to search for vacancies based on keywords so I can find positions that are relevant to my interests and skills.
  - **As a job seeker**, I want to filter openings by candidate category so I can see jobs that match my experience level.
  - **As a job seeker**, I want to filter openings by job type so I can find positions that match my career preferences.
  - **As a job seeker**, I want to filter vacancies by location so I can search for jobs in my desired area.
  - **As a job seeker**, I want to see a list of available openings along with brief information so I can quickly compare several positions.
  - **As a job seeker**, I want to know the application deadline for each vacancy so that I can manage my application time well.
  - **As a job seeker**, I want to see the full details of a vacancy so I can understand the responsibilities, qualifications, and requirements of the position.
  - **As a job seeker**, I want to apply directly through the Apply button so that the registration process is easier and more efficient.
</Accordion>
