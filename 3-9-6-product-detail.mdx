## Overview

- Displays a new page when the user clicks on one of the products on the innovation page.
- Display the main product image in large size as the main focus of the page.
- Provides a thumbnail image list on the left side to view product image variations.
- Displays breadcrumb navigation (Innovation \> Product Name) to aid user navigation.
- Display the product name clearly and prominently.
- Provide a brief description of the product as an initial overview.
- Provides product variant selection options (e.g. color or type).
- Provides product size selection options (e.g. weight: 50g, 100g, etc.).
- Provides a quantity selector (button to increase/decrease the purchase amount).
- Displays the main CTA button “Get This Product” to make a purchase.

## User Story

<Accordion title="View User Story" defaultOpen>
  - **As a visitor**, I want to see the main image of the product so that I can clearly understand the product appearance.
  - **As a visitor**, I would like to see some additional product images so I can see the product details from various angles.
  - **As a visitor**, I want to see breadcrumbs so I know where I am on the website page.
  - **As a visitor**, I want to see the product name so I know what product I'm looking at.
  - **As a visitor**, I want to see the product price along with the discount so I can consider the purchase value.
  - **As a visitor**, I want to read a short description of the product so that I get an initial idea about the product.
</Accordion>
