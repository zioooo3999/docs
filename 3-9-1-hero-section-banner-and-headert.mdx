## Overview

- Displaying the hero section as the opening page of Innovation with the main visual being the Tempe Park area from an aerial view.
- Provide a page title “Innovation” that is displayed clearly and prominently as the page identity.
- Displays navigation tabs (Our Product, Menu Inspiration, Collaboration, Inquiries) to make it easier for users to move between sections on the page.
- Using a strong visual background to give a modern, innovative and professional impression.

## User Story

<Accordion title="View User Story" defaultOpen>
  - **As a visitor**, I want to know that I am on the Innovation page, so that I understand the context of the content displayed.
  - **As a visitor**, I want to see an engaging visual display at the beginning of the page, so that I am interested in exploring further.
  - **As a visitor**, I want to be able to move between sections easily via navigation tabs, so I can go directly to the information I need.
  - **As a visitor**, I want to see a clear page structure from the start, so I don't get confused while browsing.
</Accordion>
