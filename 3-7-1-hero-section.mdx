## Overview

- Displaying a hero section as the opening page for Education with the main visual being a representative and educational image of the Tempe Park area.
- Provide a clear and prominent “Educations” page title as the page identity.
- Displays navigation tabs (History of Tempe, Fun Facts, Tempe Warrior & Hero, School Programs) to make it easier for users to move between sections.
- Using strong background visuals to give an educational, natural, and informative impression to visitors.

## User Story

<Accordion title="View User Story" defaultOpen>
  - **As a visitor**, I want to know right away that I am on the Education page, so that I understand the context of the page I am accessing.
  - **As a visitor**, I want to see an engaging visual display at the beginning of the page, so that I feel interested in exploring further.
  - **As a visitor**, I want to be able to move between educational topics easily through navigation tabs, so I can go directly to the information I need.
  - **As a visitor**, I want to see a clear page structure from the start, so I don't get confused while browsing the content.
</Accordion>
