## Overview

- Featuring the Tempe Warrior & Hero section which contains profiles of inspiring figures behind the development of tempeh.
- Presenting content in the form of a highlight profile (slider/carousel) that displays one main character at a time.
- Displaying large photos of characters as the main visual focus to increase user engagement.
- Provides information in the form of character names, section titles, and narrative descriptions that explain the character's contributions and roles.
- Provides slider navigation (arrows and indicators) to allow users to move between character profiles.
- Use a two-column layout (visual and description) to maintain balance between visual and text content.

## User Story

<Accordion title="View User Story" defaultOpen>
  - **As a visitor**, I want to know the figures who played a role in the development of tempeh, so that I can understand their contributions more deeply.
  - **As a visitor**, I want to see photos of the characters clearly displayed, so I feel more connected to the story being told.
  - **As a visitor**, I would like to read a more complete explanation of the character's role, so that I can gain richer insight.
  - **As a visitor**, I want to be able to move between characters easily using a slider, so I can explore more of the profile.
</Accordion>
