## Overview

- Displays the Facilities section for information about the supporting facilities available at Tempe Park.
- Provide informative section titles (“Feel Comfortable and Supported Throughout Your Visit”) to emphasize visitor comfort.
- Displays a list of facilities in a structured and easy-to-understand grid card format.
- Each card contains a picture of the facility, the name of the facility (such as Multi Function Room, Green Space, SME Kiosk, Café, Restaurant, Gift Shop), and a CTA “View This Place”.
- Use a grid layout (multiple columns) to display multiple features in one neat view.
- Presents representative visuals to help users understand the function of each facility.

## User Story

<Accordion title="View User Story" defaultOpen>
  - **As a user** who wants to know about the facilities at Tempe Park, I want to see a clear and structured list of the available facilities, so I can understand what facilities will support my visit.
  - **As a user**, I want to see a visual representation of each facility, so I can imagine the condition and atmosphere of the facility.
  - **As a user**, I want to press the “View This Place” button on each facility, so I can see detailed information about the facility I need.
  - **As a user**, I want to see facilities in a neat and easy-to-read layout, so I can quickly find relevant facilities.
</Accordion>
