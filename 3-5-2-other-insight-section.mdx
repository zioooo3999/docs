## Overview

- Displays the Other Insights section as a collection of additional articles in addition to the featured articles in the hero slider section.
- Provides a list of articles in list/card form with information such as thumbnail, title, date, and category.
- Provides a filter category feature (e.g. All, Sustainability, Innovation, Culture, etc.) to help users filter content.
- Provides a search bar feature to make it easier for users to find articles based on certain keywords.

## User Story

<Accordion title="View User Story" defaultOpen>
  - **As a user** who wants to read more articles, I would like to see a list of other articles available, so I can explore a variety of relevant topics.
  - **As a user**, I want to filter articles by category, so I can find content that suits my interests.
  - **As a user**, I want to search for articles using the search feature, so I can quickly find a specific article.
  - **As a user**, I want to see brief information such as the title and category of the article, so I can determine which article I want to read.
</Accordion>
