## Overview

- Displays a detail page as a continuation of the Experience page which provides more complete information regarding the attraction, activity, or facility selected by the user.
- Provide a hero section with a main visual (banner image) that represents the selected content (Attraction, Activities, or Facilities).
- Display page labels (Experience) as breadcrumbs or navigation markers so that users know the origin of the previous page.
- Displaying high-resolution images to provide an immersive feel and help users understand the atmosphere of the place or activity.

## User Story

<Accordion title="View User Story" defaultOpen>
  - **As a user** who has selected an attraction, activity, or facility, I want to see a detail page with a clear and representative key visual, so I can immediately understand the context and atmosphere of the content I selected.
  - **As a user**, I want to see navigation markers like the “Experience” label, so I know I’m still in the Experience page flow.
  - **As a user**, I want to see a consistent look and feel between detail pages, so I don't have to readapt when I open a different type of content.
</Accordion>
